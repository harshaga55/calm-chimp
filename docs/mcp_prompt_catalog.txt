Calm Chimp MCP Prompt Catalog
=============================

Use these patterns to steer the Azure + LangGraph orchestrator toward the Supabase tools exposed by the new deterministic API.

Prompt Playbook
---------------
1. Timeline insight (calendar)
   - "Show everything happening on 2025-01-20."
   - "What events do I have this weekend?"
   - "Refresh my calendar cache with the latest Supabase data."

2. Event management (calendar)
   - "Create a design review tomorrow at 10am for one hour in the Studio." → `upsert_event`
   - "Mark the planning session as completed." → `update_event_status`
   - "Cancel the retro called team-retro-42." → `delete_event`

3. Category organization (categories)
   - "Make a category named Focus Blocks using #ffb703." → `upsert_category`
   - "List all of my categories." → `list_categories`

4. Accounts & context (accounts)
   - "Who am I signed in as right now?" → `current_user_profile`

MCP Tool Reference
------------------
All tools return JSON objects aligned with the Supabase schema. Parameters are standard Python annotations.

- refresh_timeline — calendar  
  Hydrate ± cache window from Supabase and refresh the in-memory index.  
  Parameters: `anchor: str | None`

- events_for_day — calendar  
  Return cached events for a specific ISO date.  
  Parameters: `day: str`

- events_between — calendar  
  Return cached events between two ISO dates (inclusive).  
  Parameters: `start: str`, `end: str`

- upsert_event — calendar  
  Create or update an event. Missing `event_id` will generate a UUID.  
  Parameters: `title: str`, `starts_at: str`, `ends_at: str`, `event_id: str | None`, `status: str`, `category_id: str | None`, `notes: str`, `location: str | None`, `metadata: dict | None`

- update_event_status — calendar  
  Change the status (`planned`, `in_progress`, `completed`, `canceled`).  
  Parameters: `event_id: str`, `status: str`

- delete_event — calendar  
  Remove an event from Supabase and the cache.  
  Parameters: `event_id: str`

- list_categories — categories  
  List every category for the authenticated user.  
  Parameters: none

- upsert_category — categories  
  Create or update a category.  
  Parameters: `name: str`, `category_id: str | None`, `color: str | None`, `icon: str | None`, `description: str`

- delete_category — categories  
  Delete a category by id.  
  Parameters: `category_id: str`

- current_user_profile — accounts  
  Return profile data for the signed-in Supabase user.  
  Parameters: none
